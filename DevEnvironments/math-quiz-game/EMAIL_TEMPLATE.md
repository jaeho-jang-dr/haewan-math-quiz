## 📱 아이폰 설치 방법:
1. 첨부파일을 다운로드하여 압축 해제
2. index.html 파일을 Safari로 열기
3. "홈화면에 추가"를 터치해서 앱으로 설치

## 🎮 게임 특징:
- "안녕 혜완!" 맞춤 환영 메시지
- 한국어 음성 인사 및 격려
- 아이폰에 완벽 최적화된 UI
- 게임 중 메뉴 돌아가기 기능
- 선명한 성공/실패 메시지

## 🌟 특별 기능:
- 자동으로 "김혜완" 플레이어 추가
- 연속 정답 시 격려 음성: "혜완이 정말 잘하고 있어!"
- 틀렸을 때 위로 음성: "괜찮아! 혜완이는 할 수 있어!"
- PWA 지원으로 오프라인 플레이 가능
